# UK-Hospital-Operations-Optimization-
探究英国某大型骨科医院影响住院时长的原因，区分患者基础、流程因素与结构因素，以术后住院时间为核心因变量，构建多元回归模型，用于识别影响住院效率的关键机制，模拟测算优化效果，为医院周转效率与资源配置优化提供依据。


## 项目背景：

基于英国某大型骨科医院近十年（2012-2021年）逾7万条全量患者住院记录及超1万条ICD疾病编码，旨在通过数据分析厘清“临床客观规律”与“运营管理损耗”的边界，模型控制病例结构（年龄、性别、合并症、ICD分组），引入流程变量（住院类型、手术情况、等待时间）及交互项，并加入科室固定效应，探究影响住院效率的关键因素，并根据结果提出可落地的运营管理建议。

## 分析方法：

- 去除极端离群值，构建与住院时长相关的关键字段（住院时长、术后时长、预测偏差、预测时长等）。
- 对可能影响住院时长的非线性变量（如 ICD 分组、出院原因、转院地点等）进行特征工程。
- 使用相关性分析和单因素检验（非参数检验）探究住院时长的潜在影响因素。
- 对显著右偏的变量进行对数变换，以术后住院时长为因变量建立多元回归模型，并加入交互项与科室虚拟变量，考察科室间异质性；同时按科室分层回归。
- 对于存在严重缺失的住院原因变量，分别采用删除法与插补法进行稳健性分析；对合并症（0/1）进行亚组分析；因无法确认合并症与“术后并发症”之间的因果与时间顺序，依次删除相关变量及缺失字段后重新拟合模型，以检验结果稳健性。
- 对住院时长预测偏差进行下钻分析，识别实际住院时长超过预测值 3 天的患者群体，绘制其典型画像。
- 进行运营模拟分析：缩短可干预的线性变量（如术前等待时间），重新拟合模型，量化优化方案对医院运营效率及床位释放日的提升效果。


## 主要发现：

- 长尾分布：Top20%的患者锁定了**69%**的床位资源。
- 边际递减：等待时间与手术次数对住院时长的破坏力呈递减趋势（存在显著负向交互）。
- 定向施策：构建波士顿矩阵，将敏感度迥异的科室划入**排程拥堵/医疗质量/双重影响/其他因素**四大象限。
- 降本增效：模拟测算仅压缩 20% 术前等待时间，每年即可省出 4,070 个床日，流转率提升 9%。
